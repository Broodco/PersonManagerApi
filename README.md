# PersonManagerApi

PersonManagerApi est un webservice REST/JSON permettant de gérer des personnes dans une base de données.

Les fonctionalités implémentées sont : 
* des endpoints CRUD classiques
* une recherche par nom et prénom

L'application utilise .NET Core, ainsi que l'EF Core pour interagir avec une petite base de données SQLite.

Outre l'API, un autre projet contenant une suite de tests est fourni dans la solution.
